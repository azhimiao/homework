# STEM Homework examples

按 theme 分组，与 `skill-forge/built/<theme>/<skill>` 同构。

```bash
python skill-core/skill.py install examples/student-homework/homework-assistant --host cursor --scope global
```

## `student-homework/` (2)
- `homework-assistant` — 主入口
- `quartus-lab` — Quartus FPGA

## `coding-homework/` (5)
- `python-lab`, `java-oop-lab`, `ds-algorithm-lab`, `ml-notebook-lab`
- `coding-homework-assistant` — 别名

## `simulation-lab/` (4)
- `matlab-simulink-lab`, `multisim-lab`, `keil-embedded-lab`
- `simulation-homework-assistant` — 别名

重新导出：`python skill-core/scripts/export_examples.py`
